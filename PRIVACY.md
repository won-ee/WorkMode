# 개인정보처리방침 (Privacy Policy)

**WorkMode** Chrome 확장 프로그램 (이하 "본 확장 프로그램")의 개인정보처리방침입니다.

최종 수정일: 2026년 2월 20일

---

## 1. 수집하는 개인정보

본 확장 프로그램은 사용자의 개인정보를 수집하지 않습니다.

수집하지 않는 정보:
- 개인 식별 정보 (이름, 이메일, 주소 등)
- 인증 정보 (비밀번호, PIN 등)
- 위치 정보
- 웹 기록 (방문한 페이지 목록, 방문 시각 등)
- 사용자 활동 (클릭, 스크롤, 키 입력 등)
- 웹사이트 콘텐츠 (텍스트, 이미지, 동영상 등)

## 2. 데이터 저장

본 확장 프로그램은 아래 설정 데이터만 사용자의 로컬 브라우저(chrome.storage.local)에 저장합니다:

- WorkMode 활성화 상태 (ON/OFF)
- 선택한 테마
- 필터 설정 (이미지/동영상/이모지/탭 위장 ON/OFF)
- Boss Key URL
- 사이트 제외 목록

이 데이터는 사용자의 기기에만 저장되며, 외부 서버로 전송되지 않습니다.

## 3. 외부 서버 통신

본 확장 프로그램은 외부 서버와 통신하지 않습니다. 모든 기능은 사용자의 브라우저 내에서 로컬로 실행됩니다.

## 4. 제3자 데이터 공유

본 확장 프로그램은 어떠한 데이터도 제3자에게 판매, 전송, 공유하지 않습니다.

## 5. 변경 사항

본 개인정보처리방침이 변경될 경우, 이 페이지를 통해 공지합니다.

## 6. 문의

본 개인정보처리방침에 대한 문의사항이 있으시면 아래로 연락해주세요.

- GitHub Issues: https://github.com/won-ee/WorkMode/issues

## Privacy Policy (English)

**WorkMode** Chrome Extension ("the Extension") privacy policy.

Last updated: February 20, 2026

### Data Collection
This Extension does not collect any personal information or user data.

### Local Storage
The Extension stores only user preferences (activation state, selected theme, filter settings, Boss Key URL, and excluded sites) locally in the browser using chrome.storage.local. No data is transmitted to external servers.

### Third-Party Sharing
This Extension does not sell, transfer, or share any data with third parties.

### Contact
For questions about this privacy policy, please open an issue on the GitHub repository.
